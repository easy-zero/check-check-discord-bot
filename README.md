## Install  

<br />

1. 패키지 설치
   ```sh
   npm install
   ```
   <br />
2. 환경 설정
   ```sh
   cp config.json.example config.json
   vi config.json
   ```
   <br />
3. 커맨드 등록
   ```sh
   npm run deploy-commands
   ```
   <br />
4. 실행
   ```sh
   npm start
   ```
